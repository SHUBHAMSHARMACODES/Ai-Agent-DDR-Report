# 🤖 AI Agent DDR Report

An AI-powered web application that automates **Due Diligence Report (DDR)** generation using Google's Gemini AI. Users can upload company-related documents, ask questions, and generate structured reports in seconds.

---

## 🚀 Features

- 📄 AI-powered Due Diligence Report Generation
- 🤖 Gemini AI Integration
- 💬 Interactive AI Chat Interface
- 📁 Document Upload & Analysis
- ⚡ Fast and Responsive UI
- 📱 Mobile-Friendly Design

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/SHUBHAMSHARMACODES/Ai-Agent-DDR-Report.git
```

Go to the project directory

```bash
cd Ai-Agent-DDR-Report
```

Install dependencies

```bash
npm install
```

Create a `.env.local` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Start the development server

```bash
npm run dev
```
